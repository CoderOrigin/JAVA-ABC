# Mooc视频学习： 面向对象语言---JAVA

## 1.类和对象

### 1.1 对象、函数
对象 = 属性 + 服务

数据：属性，状态

操作：函数

操作是包含着数据的，把数据和对数据的操作放在一起，数据不对外面公开-->封装
### 1.2 this
在成员函数内部可以用this调用其他成员函数

### 1.3 成员变量&本地变量

本地变量：生存期和作用域都在函数内部，定义在函数内部

成员变量：生存期是对象的生成期，作用域是类中的成员函数，定义在类中

### 1.4 构造函数
构造函数与类名相同，没有返回类型，

重载：可以有多个构造函数，根据参数不同调用不同的构造函数

创建对象的时候，先调用构造函数，再初始化成员变量，再进入构造函数内部。
